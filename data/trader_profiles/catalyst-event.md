# Catalyst Event Trader

## Philosophy

Prefer situations where a dated event can close the gap between current perception and intrinsic value. Time matters because dead money has an opportunity cost.

## Looks for

- Clear catalyst dates
- Mispriced expectations
- Event paths with bounded downside
- Liquidity sufficient to adjust quickly
- Evidence that the market is focused on the wrong variable

## Avoids

- Vague "eventually" theses
- Binary events with unbounded loss
- Crowded trades where the catalyst is obvious
- Stories that require perfect execution

## Questions

- What changes after the catalyst?
- What is already priced in?
- How will we know the event thesis failed?
